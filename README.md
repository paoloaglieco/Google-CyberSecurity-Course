# Google Cybersecurity Professional Certificate

Materiali, esercizi e progetti del **Google Cybersecurity Professional Certificate** su Coursera.

## Moduli

- **Security Audit** — conduzione di audit di sicurezza su infrastrutture aziendali

## Argomenti trattati

- Fondamenti di cybersecurity e gestione del rischio
- Network security e analisi del traffico
- Linux e strumenti a riga di comando
- SQL per analisi di sicurezza
- Asset management e vulnerability assessment
- Detection and response
- Python per la sicurezza informatica
